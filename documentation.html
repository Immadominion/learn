Submission for the Solana Superteam UK Bounty: Build a Product Using The Grid API.

Overview

The Grid is a startup that aggregates data from various Solana ecosystem products, teams, and companies, making it easier for developers to access this information. Potential applications integrating with The Grid include wallets, DEXs, data platforms, ecosystem pages, token lists, block explorers, retail platforms, exchanges, and compliance tools.

Problem Statement

Accessing comprehensive and organized data from the Solana ecosystem can be challenging for developers. This bounty aims to encourage the development of innovative products that leverage The Grid API to solve real-world problems and enhance the Solana ecosystem.

For moderators, it might be hard to keep track of XPs per event or distribute USDC after a Kahoot session in an ecosystem call.

Solution

We developed a quiz app where users can choose quiz topics that query The Grid API, using Gemini to generate the questions. This product showcases innovation, completeness, and relevance to current market needs.

Installation

git clone https://github.com/your-repo/quiz-app.git
cd quiz-app
flutter pub get
flutter run

Usage

Open the app, select a quiz topic, and start answering questions. The app will fetch questions from The Grid API and present them in a user-friendly format.

Features

  • Fetches real-time quiz questions from The Grid API.
  • Generates questions using Gemini.
  • Interactive UI with progress tracking.
  • Leaderboard to track top scorers.

API Integration

The app integrates with The Grid API to fetch quiz questions and other relevant data. For more information on The Grid API, visit The Grid Explorer.

Market Fit Analysis

This quiz app addresses the need for educational and interactive content within the Solana ecosystem. By leveraging The Grid API, we provide users with up-to-date information and an engaging way to learn about Solana-related topics.
